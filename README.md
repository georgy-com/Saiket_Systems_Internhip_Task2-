# Saiket_Systems_Internhip_Task2-
This Repo contains Task2 (Stock Portfolio Tracker)

Project Overview
This project is a simple Stock Portfolio Tracker developed using Python.
It allows users to input stock symbols and the number of shares purchased, then calculates the total value of their investment.

The program also saves a summary of the portfolio to a .txt file for future reference.

Features
Accepts user input for stock symbols and quantities
Validates stock availability
Calculates total investment value
Stores portfolio data dynamically
Exports results to a text file (portfolio_summary.txt)
Supported Stocks

The system uses hardcoded stock prices for the following companies:

AAPL (Apple)
TSLA (Tesla)
GOOGL (Google)
AMZN (Amazon)

How It Works
The program prompts the user to enter a stock symbol
The user inputs the number of shares
The system validates the input
The total value is calculated
The process repeats until the user types done
A summary of the portfolio is displayed and saved
